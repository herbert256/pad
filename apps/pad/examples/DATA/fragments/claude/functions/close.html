The close function outputs a literal closing brace }.

Basic Usage

{close}

Result: }

Document Closing Tags

{open}/if{close}

Shows: {/if}

{open}/users{close}

Shows: {/users}

Complete Tag Documentation

{open}if $x gt 0{close}
  positive
{open}/if{close}

Shows:

{if $x gt 0}
  positive
{/if}

Note

Always use open and close together when documenting PAD syntax. They escape the braces so PAD displays them literally instead of parsing them.
