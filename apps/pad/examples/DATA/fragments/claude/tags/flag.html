The flag tag sets a flag during iteration.

Basic Flag

{sequence 5, name='i'}
  {if $i eq 3}
    {flag 'found'}
  {/if}
{/sequence}
{if $found}Found 3!{/if}

Flag for Detection

Flags are useful to detect if a condition was met during iteration.

{items}
  {if $status eq 'error'}
    {flag 'hasErrors'}
  {/if}
{/items}
{if $hasErrors}
  Warning: Errors detected!
{/if}
