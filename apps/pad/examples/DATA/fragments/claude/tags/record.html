The record tag retrieves a single row from the database.

Syntax

{record "columns from table where condition"}
  {$column1} - {$column2}
{/record}

Example: User Profile

{record "* from users where id = 1"}
  <h2>{$name}</h2>
  <p>Email: {$email}</p>
  <p>Role: {$role}</p>
{else}
  <p>User not found</p>
{/record}

PHP Variable Method

// In PHP:
$user = db("record * from users where id = 1");

// In template:
{user}
  Name: {$name}
{else}
  Not found
{/user}

Record vs Array

TagReturnsUse Case
{record}First row onlySingle item lookup
{array}All matching rowsLists, tables
