The false tag never executes its content (condition always false).

Basic False

{false}
  This never displays
{/false}

(Nothing shown above)

False with Else

{false}
  Never shown
{else}
  Always shown
{/false}

Always shown (else always executes)

Use Case: Disabled Code

{false}
  {dump}
  Old debug code that's disabled
{/false}

(Debug code above is disabled)
