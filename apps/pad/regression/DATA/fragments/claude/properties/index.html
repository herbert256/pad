Tag properties provide contextual information during template execution using @ syntax.

All Properties

  • first - TRUE if first iteration (Iteration State)
  • last - TRUE if last iteration (Iteration State)
  • notFirst - TRUE if NOT first iteration (Iteration State)
  • notLast - TRUE if NOT last iteration (Iteration State)
  • border - TRUE if first OR last (Iteration State)
  • middle - TRUE if neither first nor last (Iteration State)
  • even - TRUE if even occurrence (Iteration State)
  • odd - TRUE if odd occurrence (Iteration State)
  • current - Current occurrence number (1-based) (Counter)
  • count - Total number of items (Counter)
  • remaining - Items remaining after current (Counter)
  • done - Items completed before current (Counter)
  • data - Complete data array (Data Access)
  • key - Current array key (Data Access)
  • keys - All keys with values (iterable) (Data Access)
  • fields - Field name/value pairs (iterable) (Data Access)
  • firstFieldName - Name of first field (Data Access)
  • firstFieldValue - Value of first field (Data Access)
  • name - Name of current tag (Tag Metadata)
  • parameter - Named parameter value (Parameters)
  • parameters - All named parameters (iterable) (Parameters)
  • option - Positional option value (Parameters)
  • options - All positional options (iterable) (Parameters)
  • variable - Level-scoped variable value (Variables)
  • variables - All level variables (iterable) (Variables)
