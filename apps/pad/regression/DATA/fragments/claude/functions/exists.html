The exists function tests if a file exists.

Basic Usage 

{if {echo 'index.php' | exists} eq 1}File found{/if}

Result: index.php exists

Check Path 

{if {echo '_lib/helpers.php' | exists} eq 1}Helper exists{/if}

Not found: Helper file missing

Conditional Include 

{if {echo 'custom.pad' | exists} eq 1}
  {custom}
{/if}

Chained Example 

{data $file = 'readme.txt'}
{if {echo $file | exists} eq 1}Found{/if}

Note

Returns 1 (true) if file exists, 0 (false) otherwise. Path is relative to current page.
