The now function returns the current Unix timestamp.

Basic Usage

{now}

Timestamp: 1765931828

Format Current Time

{echo {now} | date('Y-m-d')}

Today: 2025-12-17

Full DateTime

{echo {now} | date('Y-m-d H:i:s')}

Now: 2025-12-17 00:37:08

Chained Examples

{echo {now} | date('l, F j, Y')}

Full date: Wednesday, December 17, 2025

{echo {now} | date('H:i')}

Time: 00:37

Note

Returns Unix timestamp (seconds since 1970-01-01). Pipe to date for formatting.
