# Check Application

Comprehensive test suite for validating PAD framework features and functionality.

## Structure

```
check/
├── _data/           # Test data files
├── _functions/      # Test pipe functions
├── _include/        # Test includes
├── _lib/            # Test library functions
├── _options/        # Test options
├── _scripts/        # Test shell scripts
├── _tags/           # Test custom tags
├── control/         # Control flow tests
├── db/              # Database operation tests
├── deep/            # Deep nesting tests
├── error/           # Error handling tests
├── file/            # File operation tests
├── fragments/       # Test fragments
├── functions/       # Pipe function tests
├── hello/           # Basic hello tests
├── miscellaneous/   # Miscellaneous tests
├── options/         # Tag option tests
├── select/          # PAD Select tests
├── start/           # Startup tests
├── tableFun/        # Table function tests
├── tags/            # Tag tests
└── vars/            # Variable tests
```

## Purpose

Tests various PAD features including:
- Control flow (`if`, `case`, loops)
- Database operations
- File handling
- Custom tags and functions
- Error handling
- Deep template nesting

## Access

Via web browser: `http://server/check/`
