The keep tag marks an item to keep in filtered results.

Keep Syntax 

{items filter}
  {if $value gt 10}
    {keep}
  {/if}
{/items}

Filter Example 

{data 'values'}[5, 15, 8, 20, 3, 12]{/data}
{values}
  {if $values gt 10}{keep}{/if}
  {$values}
{/values}

All values: 5 15 8 20 3 12

Note

The keep tag is used with filter options to build filtered result sets.
