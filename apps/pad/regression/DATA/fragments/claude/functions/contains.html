The contains function tests if a value contains a substring (returns 1 or empty).

Basic Usage 

{'hello world' | contains('world')}

Contains 'world': 'hello world' | contains('world') (1 = true) 

{'hello world' | contains('foo')}

Contains 'foo': 'hello world' | contains('foo') (empty = false)

Conditional Display 

{if $sampleEmail | contains('@')}
  Valid email format
{/if}
