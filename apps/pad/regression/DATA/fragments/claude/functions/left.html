The left function returns the first N characters.

Basic Usage 

{'Hello World' | left(5)}

First 5: 'Hello World' | left(5)

Truncate with Ellipsis 

{'This is a very long text that needs truncation' | left(20)}...

Result: 'This is a very long text that needs truncation' | left(20)...

Chained Examples 

{$sampleEmail | before('@') | left(1) | upper}

First initial: J 

{'HELLO WORLD' | left(5) | lower | capitalize}

Transform first word: 'HELLO WORLD' | left(5) | lower | capitalize

Complex Chain 

{users}
  {$name | left(1)}. {$name | after(' ')}
{/users}

Abbreviated names:

A. Johnson
B. Smith
C. Brown
