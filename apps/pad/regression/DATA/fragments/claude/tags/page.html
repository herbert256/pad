The page tag includes another page.

Include a Subpage

{page 'fragments/claude/subpage'}

Include with Variable Passing

{page 'fragments/claude/subpage', $subpageMessage='Custom message'}

Include Snippets from _include

{greeting}

Hello from included snippet!

{snippet}
Snippet: This is reusable content from _include/snippet.pad

Scope Options

{page 'path', sandbox}  {!-- isolated scope --}
{page 'path', clean}    {!-- clean scope --}

These options control variable scope when including pages.
