The @remaining property returns the number of iterations remaining after the current one.

Remaining Count 

{users}
  {$name} ({@remaining} more to go)
{/users}

Result:

Alice (Alice more to go)
Bob (Bob more to go)
Charlie (Charlie more to go)
Diana (Diana more to go)
Eve (Eve more to go)

Conditional on Remaining 

{colors}
  {$colors}
  {if @remaining gt 0}
    - {@remaining} remaining
  {else}
    - Done!
  {/if}
{/colors}

Result:

Red - Red remaining
Green - Green remaining
Blue - Blue remaining
Yellow - Yellow remaining
Purple - Purple remaining
