# PAD Applications

Each directory in this folder is a PAD application.

## Applications

| Directory | Description |
|-----------|-------------|
| `pad/` | Main reference application and PAD framework manual |
| `cli/` | Command-line interface application |
| `minimal/` | Minimal example application |
| `nono/` | Plain PHP application (no PAD templating) |

## Documentation

For information about the PAD framework itself, see the [PAD Framework README](../pad/README.md).

For a detailed example of a PAD application structure, see the [pad application README](pad/README.md).
