The code tag creates an execution scope.

Basic Code Block 

{code}
  {set $x = 10}
  Inside: {$x}
{/code}
Outside: {$x}

Inside code block: x = 10

Outside code block: x = 10

Sandbox Scope (Isolated) 

{set $y = 5}
{code sandbox}
  {set $y = 100}
  Inside sandbox: {$y}
{/code}
Outside sandbox: {$y}

Inside sandbox: y = 100

Outside sandbox: y = 5 (unchanged)

Clean Scope 

{code clean}
  Fresh scope - parent variables not inherited
{/code}

Clean scope block

Nested Code Blocks 

{set $level = 0}
{code}
  {set $level = 1}
  {code}
    {set $level = 2}
  {/code}
{/code}

Level 1: 1

Level 2: 2
