The while tag loops while a condition is true.

Basic While Loop

{set $i = 1}
{while $i le 5}
  Item {$i}
  {increment $i}
{/while}

While with Decrement

{set $countdown = 5}
{while $countdown gt 0}
  {$countdown}...
  {decrement $countdown}
{/while}

Nested While Loops

{set $row = 1}
{while $row le 3}
  {set $col = 1}
  {while $col le 4}
    ({$row},{$col})
    {increment $col}
  {/while}
  {increment $row}
{/while}
