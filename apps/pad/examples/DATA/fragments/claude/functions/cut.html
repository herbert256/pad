The cut function removes all occurrences of a substring.

Basic Usage

{echo 'hello-world' | cut('-')}

Result: helloworld

Remove Prefix

{echo 'prefix_name' | cut('prefix_')}

Without prefix: name

Remove Characters

{echo 'a.b.c.d' | cut('.')}

No dots: abcd

Chained Examples

{echo 'Mr. John Smith' | cut('Mr. ') | upper}

Name only: JOHN SMITH

{echo '123-456-7890' | cut('-') | left(6)}

First 6 digits: 123456

{echo 'test_file_name.txt' | cut('_') | beforeLast('.')}

Clean name: testfilename

Note

The cut function is shorthand for replace(text, '').
