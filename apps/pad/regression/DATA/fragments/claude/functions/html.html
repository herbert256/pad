The html function escapes HTML special characters for safe display.

Basic Usage 

{echo $htmlText | html}

Result: <p>Hello & "World"</p>

Escape Tags 

{echo '
' | html}

Safe: <script>alert("XSS")</script>

Escape Quotes 

{echo 'He said "hello"' | html}

Escaped: He said "hello"

Chained Examples 

{echo $htmlText | html | replace('&', '+')}

Modified: <p>Hello + "World"</p> 

{echo 'Bold' | html | upper}

Escaped and upper: <B>BOLD</B>

Characters Escaped

  • < becomes &lt;
  • > becomes &gt;
  • & becomes &amp;
  • " becomes &quot;

Note

Always escape user input before displaying in HTML to prevent XSS attacks.
