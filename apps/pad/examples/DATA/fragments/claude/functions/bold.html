The bold function wraps text in HTML <b> tags.

Basic Usage

{echo 'important' | bold}

Result: important

Emphasize Text

{echo 'warning' | bold}

Warning: warning

Chained Examples

{echo 'hello' | upper | bold}

Loud: HELLO

{echo $sampleText | left(5) | bold}

First word: Hello

{echo 'status: ok' | after(': ') | capitalize | bold}

Status: Ok

Note

Outputs HTML tags directly. The text will appear bold in HTML rendering.
