The time function is an alias for date.

Basic Usage 

{now | time('H:i:s')}

Current time: 21:06:00

Same as Date 

{now | date('Y-m-d H:i')}

Date: 2025-12-14 21:06 

{now | time('Y-m-d H:i')}

Time: 2025-12-14 21:06

Time Formats 

{now | time('H:i')}

Hours:minutes: 21:06 

{now | time('g:i a')}

12-hour: 9:06 pm 

{now | time('H')}

Hour only: 21

With Modifier 

{now | time('H:i', '+2 hours')}

In 2 hours: 23:06

Note

Use time or date interchangeably based on readability. Some prefer time when focusing on time-of-day formatting.
