The right function returns the last N characters of a string.

Basic Usage

{echo $sampleText | right(10)}

Last 10: ample Text

File Extension

{echo 'document.txt' | right(3)}

Extension: txt

Last Digits

{echo '1234567890' | right(4)}

Last 4: 7890

Chained Examples

{echo $sampleText | right(15) | upper}

Last 15 uppercase: LD EXAMPLE TEXT

{echo $filePath | right(10) | lower}

End of path: readme.txt

{echo 'HELLO WORLD' | right(5) | lower | capitalize}

Last word formatted: World

Note

If the string is shorter than N, the entire string is returned.
