The left function returns the first N characters of a string.

Basic Usage 

{echo $sampleText | left(10)}

First 10: Hello Worl

Abbreviation 

{echo 'Documentation' | left(3)}

Abbrev: Doc

Initial 

{echo 'John' | left(1)}

Initial: J

Chained Examples 

{echo $sampleText | left(20) | upper}

First 20 uppercase: HELLO WORLD EXAMPLE 

{echo $email | before('@') | left(5) | upper}

Username prefix: JOHN. 

{echo 'Hello World' | left(5) | trim}

Trimmed left: Hello

Note

If the string is shorter than N, the entire string is returned.
