The field tag retrieves a single field value from the database.

Syntax 

{field "column from table where condition"}

Examples 

{field "name from users where id = 1"}
{field "count(*) from orders"}
{field "email from users where username = 'admin'"}

PHP Variable Method 

// In PHP:
$userName = db("field name from users where id = 1");

// In template:
{$userName}

Direct Template Method 

User count: {field "count(*) from users"}

Note

Returns a single scalar value from the first matching row.

Requires database connection configured in $padSql* variables.
