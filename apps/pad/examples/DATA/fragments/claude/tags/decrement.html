The decrement tag subtracts 1 from a variable.

Basic Decrement

{set $counter = 5}
Counter: {$counter}
{decrement $counter}
Counter: {$counter}
{decrement $counter}
Counter: {$counter}
  • Counter: 5
  • Counter: 4
  • Counter: 3

Countdown

{set $n = 5}
{while $n gt 0}
  {$n}...
  {decrement $n}
{/while}
Go!

5... 4... 3... 2... 1... Go!

Decrement to Negative

{set $x = 2}
  • x = 2
  • x = 1
  • x = 0
  • x = -1
