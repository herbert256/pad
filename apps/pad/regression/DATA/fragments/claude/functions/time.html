The time function is an alias for date.

Basic Usage 

{now | time('H:i:s')}

Current time: 23:42:02

Same as Date 

{now | date('Y-m-d H:i')}

Date: 2025-12-14 23:42 

{now | time('Y-m-d H:i')}

Time: 2025-12-14 23:42

Time Formats 

{now | time('H:i')}

Hours:minutes: 23:42 

{now | time('g:i a')}

12-hour: 11:42 pm 

{now | time('H')}

Hour only: 23

With Modifier 

{now | time('H:i', '+2 hours')}

In 2 hours: 01:42

Note

Use time or date interchangeably based on readability. Some prefer time when focusing on time-of-day formatting.
