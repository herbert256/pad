The now function returns the current Unix timestamp.

Basic Usage 

{now}

Timestamp: 1766239973

Format Current Time 

{echo {now} | date('Y-m-d')}

Today: 2025-12-20

Full DateTime 

{echo {now} | date('Y-m-d H:i:s')}

Now: 2025-12-20 14:12:53

Chained Examples 

{echo {now} | date('l, F j, Y')}

Full date: Saturday, December 20, 2025 

{echo {now} | date('H:i')}

Time: 14:12

Note

Returns Unix timestamp (seconds since 1970-01-01). Pipe to date for formatting.
