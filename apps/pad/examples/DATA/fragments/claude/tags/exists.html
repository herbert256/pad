The exists tag checks if a file exists.

Check File Exists 

{exists APP . 'fragments/claude/index.pad'}
  File exists
{else}
  File not found
{/exists}

File exists: fragments/claude/index.pad

Check Non-Existent File 

{exists APP . 'fragments/claude/nonexistent.pad'}
  Found
{else}
  Not found
{/exists}

Not found: fragments/claude/nonexistent.pad

Note

The exists tag uses APP constant for application-relative paths.
