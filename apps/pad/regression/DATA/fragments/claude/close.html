The close tag outputs the closing brace character.

Basic Usage 

{close}

Result: }

Combine with Open 

{open}content{close}

Result: {content}

Empty Braces 

{open}{close}

Result: {}

JSON-like Output 

{open}"name": "value"{close}

Result: {"name": "value"}

Practical Examples

Description Code Result
Variable {open}$var{close} {$var}
If tag {open}if{close} {if}
Close tag {open}/if{close} {/if}
With param {open}set x=1{close} {set x=1}

Note

The open and close tags are essential for documenting PAD syntax. Without them, the template engine would try to process the example code.
