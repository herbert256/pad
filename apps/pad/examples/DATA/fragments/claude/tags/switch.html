The switch tag alternates between values on each call.

Basic Switch (Two Values)

{switch 'odd', 'even'}

Called 6 times:

  • odd
  • even
  • odd
  • even
  • odd
  • even

Switch for Alternating Row Colors

<tr style="background:{switch '#f0f0f0', '#e0e0e0'}">
Row 1
Row 2
Row 3
Row 4

Switch with Multiple Values

{switch 'A', 'B', 'C'}

Cycles through A, B, C:

  • A
  • B
  • C
  • A
  • B
  • C
