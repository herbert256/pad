## Testing

The PAD reference application includes a regression test system:

- Location: `apps/pad/regression/`
- Run via browser at `/regression`
- Compares rendered output against stored reference files

## Application Manual

The `apps/pad/` directory serves as both documentation and a working test suite with fragments covering all PAD features.
