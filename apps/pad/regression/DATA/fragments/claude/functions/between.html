The between function tests if a value is between two values (exclusive).

Basic Usage 

{if {echo 5 | between(1, 10)} eq 1}In range{/if}

Result: 5 is between 1 and 10

Edge Cases (Exclusive) 

{if {echo 10 | between(1, 10)} eq 0}At boundary{/if}

Outside Range 

{if {echo 15 | between(1, 10)} eq 0}Out of range{/if}

String Comparison 

{if {echo 'b' | between('a', 'c')} eq 1}In range{/if}

Result: 'b' is between 'a' and 'c'

Note

Exclusive: boundaries are NOT included. Use range for inclusive comparison.
