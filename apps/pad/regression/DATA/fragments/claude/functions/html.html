The html function escapes HTML special characters for safe display.

Basic Usage 

{$sampleHtml | html}

Original: <script>alert("XSS")</script>

Escaped: <script>alert("XSS")</script>

Prevent XSS 

{'<img src=x onerror=alert(1)>' | html}

Safe output: '' | html

Chained Examples 

{'  <b>Bold</b>  ' | trim | html}

Trimmed and escaped: ' Bold ' | trim | html 

{'<script>' | html | upper}

Escaped then uppercase: '
