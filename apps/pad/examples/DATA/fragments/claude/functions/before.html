The before function returns everything before the first occurrence of a delimiter.

Basic Usage

{echo $email | before('@')}

Username: john.smith

Extract Protocol

{echo 'https://example.com' | before('://')}

Protocol: https

First Name

{echo 'John Smith' | before(' ')}

First name: John

Chained Examples

{echo $email | before('@') | upper}

Username uppercase: JOHN.SMITH

{echo $email | before('@') | replace('.', ' ') | capitalize}

Formatted name: John Smith

{echo 'file.name.txt' | before('.') | upper}

First part uppercase: FILE

Note

If the delimiter is not found, the original string is returned unchanged.
