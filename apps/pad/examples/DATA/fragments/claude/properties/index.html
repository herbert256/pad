PAD Tag Properties

Access iteration state within loops using {property:name}...{/property:name} syntax.

Available Properties

  • first - First iteration
  • last - Last iteration
  • notFirst - Not first
  • notLast - Not last
  • border - First or last
  • middle - Neither first nor last
  • even - Even iteration
  • odd - Odd iteration
  • current - Current index
  • count - Total count
  • remaining - Items remaining
  • done - Items done
  • key - Array key
  • name - Tag name
