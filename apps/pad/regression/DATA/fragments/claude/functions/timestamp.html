The timestamp function is an alias for date.

Basic Usage 

{now | timestamp('Y-m-d H:i:s')}

Full timestamp: 2025-12-14 23:42:02

Same as Date 

{now | date('c')}

Date (ISO 8601): 2025-12-14T23:42:02+00:00 

{now | timestamp('c')}

Timestamp (ISO 8601): 2025-12-14T23:42:02+00:00

Database Formats 

{now | timestamp('Y-m-d H:i:s')}

MySQL datetime: 2025-12-14 23:42:02 

{now | timestamp('U')}

Unix timestamp: 1765755722

With Modifier 

{now | timestamp('Y-m-d H:i:s', '+1 day')}

Tomorrow same time: 2025-12-15 23:42:02

Note

timestamp, time, and date are all identical. Use whichever is most readable in context.
