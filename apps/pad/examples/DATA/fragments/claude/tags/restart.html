The restart tag restarts page processing.

Syntax

{restart}

Use Case

{action 'process'}
  {!-- Process data --}
  {set $message = 'Done!'}
  {restart}  {!-- Restart to show message --}
{/action}

{if $message}
  {$message}
{/if}

Note

Restart re-executes the page from the beginning with current state.

Useful after form processing to show updated data.
