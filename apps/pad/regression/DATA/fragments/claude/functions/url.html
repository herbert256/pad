The url function URL-encodes a value for use in URLs.

Basic Usage 

{'hello world' | url}

Result: 'hello world' | url

Special Characters 

{'name=John&age=30' | url}

Encoded: 'name=John&age=30' | url

Build URLs 

?search={'search term here' | url}

URL: ?search='search term here' | url

Chained Examples 

{'  My Search Query  ' | trim | url}

Clean and encode: ' My Search Query ' | trim | url 

{'Category / Subcategory' | replace('/', '-') | url}

Safe path: 'Category / Subcategory' | replace('/', '-') | url

Dynamic Links 

{users}
  <a href="?user={$name | url}">{$name}</a>
{/users}

User links:

Alice Johnson
Bob Smith
Charlie Brown
