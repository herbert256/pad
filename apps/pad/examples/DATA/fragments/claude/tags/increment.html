The increment tag adds 1 to a variable.

Basic Increment

{set $counter = 0}
Counter: {$counter}
{increment $counter}
Counter: {$counter}
{increment $counter}
Counter: {$counter}
  • Counter: 0
  • Counter: 1
  • Counter: 2

Increment in Loop

{set $i = 1}
{while $i le 5}
  {$i}
  {increment $i}
{/while}

1 2 3 4 5

Multiple Counters

{set $a = 0}
{set $b = 10}
Loop:
  a={$a}, b={$b}
  {increment $a}
  {decrement $b}
