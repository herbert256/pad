The action tag executes a sequence action.

Syntax 

{action 'action_name'}

What Are Actions?

Actions are predefined operations within the PAD sequence system. They allow you to trigger specific behaviors during sequence processing.

Example Usage 

{action 'process'}

This would execute the 'process' action defined in the sequence system.

Common Actions

Action Description
process Process sequence data
filter Apply filters to data
transform Transform data values

Use with Sequences 

{sequence '1..10'}
  {action 'validate'}
  {$value}
{/sequence}

Use Cases

  • Trigger sequence operations
  • Execute predefined data transformations
  • Call sequence system hooks

Note

The action tag is part of PAD's sequence subsystem. Actions must be defined within the sequence system to be executed.
