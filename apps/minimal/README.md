# Minimal Application

The absolute minimum PAD application - a single template file with no PHP.

## Files

| File | Description |
|------|-------------|
| `index.pad` | Single template file |

## Code

**index.pad**:
```
Hello world
```

## Concept

Demonstrates that PAD applications don't require a `.php` file. A `.pad` template alone is sufficient for static content.

## Access

Via web browser: `http://server/minimal/`
