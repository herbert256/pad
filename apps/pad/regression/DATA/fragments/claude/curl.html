The curl tag makes HTTP requests.

Syntax 

{curl 'http://example.com'}

Basic GET Request 

{curl 'http://example.com/api/data'}

Makes a GET request to the URL and returns the response.

With Parameters 

{curl 'http://example.com/api', id=123, format='json'}

Parameters are added to the query string.

Self-Referencing URL 

{curl 'SELF://api/endpoint'}

The SELF:// prefix is replaced with the current host.

Options

Option Description
url (first param) The URL to request
method HTTP method (GET, POST, etc.)
data Request body data
Additional params Added to query string

Example: Fetch API Data 

{curl 'https://api.example.com/users', limit=10}

Example: POST Request 

{curl 'https://api.example.com/submit', method='POST', data='payload'}

Error Handling

The curl tag throws an error if the HTTP response is not 200 OK. Wrap in error handling if needed.

Use Cases

  • Fetch data from external APIs
  • Internal service communication
  • Webhook triggers
  • Data synchronization

Note

The curl tag is for server-side HTTP requests. For client-side requests, use JavaScript/AJAX.
