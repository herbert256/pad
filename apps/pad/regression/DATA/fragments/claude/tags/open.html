The open tag outputs a literal opening brace.

Escaping PAD Syntax 

To show {$variable} literally:
{open}$variable{close}

Result: {$variable}

Documenting PAD Code 

{open}if $count gt 0{close}
  Has items
{open}/if{close}

Shows:

{if $count gt 0}
Has items
{/if}

Multiple Braces 

{open}{open}nested{close}{close}

Result: {{nested}}
