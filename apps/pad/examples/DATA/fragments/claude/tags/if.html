The if tag provides conditional execution.

Basic If/Else

{if $count gt 0}
  Has items
{else}
  Empty
{/if}

Has items (count = 5)

Comparison Operators

OperatorMeaningExampleResult
eq / ==Equal5 eq 5true
ne / !=Not equal5 ne 3true
gt / >Greater than5 gt 3true

ge / >=Greater or equal5 ge 5true

If/Elseif/Else

{if $score ge 90}
  Grade: A
{elseif $score ge 80}
  Grade: B
{elseif $score ge 70}
  Grade: C
{else}
  Grade: F
{/if}

Grade: B (score = 85)

Logical Operators

{if $role eq 'admin' and $count gt 0}
  Admin with items
{/if}

Admin with items

{if $role eq 'guest' or $role eq 'admin'}
  Guest or Admin
{/if}

Guest or Admin (role = admin)

String Comparison

{if $name eq 'Alice'}
  Hello Alice!
{/if}

Hello Alice!
