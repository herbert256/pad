The case tag provides multi-branch conditional execution using when.

Basic Case Statement

{case 2}
  {when 1} One
  {when 2} Two
  {when 3} Three
  {else} Default
{/case}
Two

Case with Variable

{set $day = 3}
{case $day}
  {when 1} Monday
  {when 2} Tuesday
  {when 3} Wednesday
  {when 4} Thursday
  {when 5} Friday
  {else} Weekend
{/case}
Wednesday

Case with Strings

{set $color = 'green'}
{case $color}
  {when 'red'} Stop
  {when 'yellow'} Caution
  {when 'green'} Go
  {else} Unknown
{/case}
Go
