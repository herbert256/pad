The sequence sub-system of PAD, many many ways to get a sequence of numbers.

There are 4 basic Sequence concepts

sequences Something that defines a sequence list
stores A stored sequence list
actions An operation executed on a sequence list
ones An operation that reduces a sequence list to a single value
