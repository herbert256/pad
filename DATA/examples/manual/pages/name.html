Name

Every PAD tag gets a name, generated by PAD.

- Most of the time it is just the tag name
- Or the first parameter ( for example with the tags {content} & {data} )
- Or something else PAD thinks it is smart to use as name.

It is possible to override the name that PAD generates with the name option.

{padname='myTag'}
