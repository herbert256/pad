The true tag returns the boolean TRUE value.

Basic Usage 

{true}

Result:

In Conditions 

{if {true} eq TRUE}
  This is shown because true equals TRUE
{/if}

Set Variable to True 

{set isActive={true}}
{if $isActive eq TRUE}Active{else}Inactive{/if}

Status: Inactive

Compare with FALSE 

{if {true} ne FALSE}
  TRUE is not equal to FALSE
{/if}

Boolean Values Table

Tag Value Type
{true} Boolean
{false} Boolean
TRUE (literal) TRUE Boolean
FALSE (literal) FALSE Boolean

Use Case: Default Value 

{set showHeader={true}}
{if $showHeader eq TRUE}
  <header>Header Content</header>
{/if}
