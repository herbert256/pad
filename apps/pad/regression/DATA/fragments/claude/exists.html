The exists tag checks if a file exists.

Check File Exists 

{exists APP . 'index.pad'}
  index.pad exists
{else}
  index.pad not found
{/exists}

index.pad exists in APP directory

Check Non-Existent File 

{exists APP . 'nonexistent.xyz'}
  File exists
{else}
  File does not exist
{/exists}

File does not exist

Check Directory 

{exists APP . 'fragments/claude'}
  claude directory exists
{/exists}

claude directory exists

Conditional Include 

{exists APP . 'optional.pad'}
  {page 'optional'}
{else}
  Optional content not available
{/exists}

Optional content not available

Check Config File 

{exists APP . '_config/config.php'}
  Custom configuration loaded
{else}
  Using default configuration
{/exists}

Custom configuration available

Use Cases

  • Check if optional files exist before including
  • Verify uploads or generated files
  • Feature detection based on file presence
  • Conditional loading of configurations
