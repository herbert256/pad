The true tag always executes its content (condition always true).

Basic True

{true}
  This always displays
{/true}

This always displays

True with Else

{true}
  True branch
{else}
  Never shown
{/true}

True branch (else never executes)

Nested True for Scoping

{true}
  {set $local = 'inner value'}
  Inside: {$local}
{/true}

Inside: inner value
