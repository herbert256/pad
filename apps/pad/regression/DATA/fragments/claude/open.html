The open tag outputs the opening brace character.

Basic Usage 

{open}

Result: {

Show Variable Syntax 

{open}$variableName{close}

Result: {$variableName}

Show Tag Syntax 

{open}tagname{close}

Result: {tagname}

Document PAD Syntax 

To show a variable, use {open}$name{close}

Result: To show a variable, use {$name}

Show Complete Tag 

{open}if $count > 0{close}...{open}/if{close}

Result: {if $count > 0}...{/if}

Use in Documentation

The open/close tags are commonly used throughout this documentation to show PAD syntax examples without the tags being processed.

Tag Output
{open} {
{close} }
