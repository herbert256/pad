# tags/

Built-in PAD tags that provide control flow, data manipulation, and utility functions.

## Control Flow
- `if.php` - Conditional with `{elseif}` support
- `while.php` / `until.php` - Loop constructs
- `switch.php` / `case.php` - Switch/case logic
- `check.php` - Condition checking
- `exists.php` - Existence check

## Data Tags
- `data.php` - Data iteration
- `array.php` - Array iteration
- `record.php` - Single record access
- `field.php` - Field access
- `set.php` - Variable setting
- `get.php` - Variable getting

## Output Tags
- `echo.php` - Output value
- `output.php` - Output handling
- `content.php` - Content manipulation
- `dump.php` - Debug dump
- `tidy.php` - HTML tidy

## File/External
- `file.php` / `files.php` - File operations
- `dir.php` - Directory operations
- `curl.php` - HTTP requests
- `page.php` - Page inclusion
- `redirect.php` - HTTP redirect
- `ajax.php` - AJAX handling

## Control
- `exit.php` - Exit processing
- `restart.php` - Restart processing
- `continue.php` - Continue loop
- `error.php` / `exception.php` - Error handling
- `ignore.php` - Ignore content
- `sandbox.php` - Sandboxed execution

## State
- `open.php` / `close.php` - Scope markers
- `true.php` / `false.php` / `null.php` - Constant values
- `increment.php` / `decrement.php` - Counter operations
- `count.php` - Counting

## Subdirectory
- `go/` - Tag execution helpers (store.php, etc.)
