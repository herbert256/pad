The now function returns the current Unix timestamp.

Basic Usage 

{now}

Timestamp: 1765864262

Format Current Time 

{echo {now} | date('Y-m-d')}

Today: 2025-12-16

Full DateTime 

{echo {now} | date('Y-m-d H:i:s')}

Now: 2025-12-16 05:51:02

Chained Examples 

{echo {now} | date('l, F j, Y')}

Full date: Tuesday, December 16, 2025 

{echo {now} | date('H:i')}

Time: 05:51

Note

Returns Unix timestamp (seconds since 1970-01-01). Pipe to date for formatting.
