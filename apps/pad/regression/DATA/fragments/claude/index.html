Working examples for all PAD template tags. Click a tag name to see examples.

All Tags

  • if - Conditional execution (Control Flow)
  • case - Multi-branch conditional (Control Flow)
  • switch - Alternating values (Control Flow)
  • while - Loop while condition true (Control Flow)
  • until - Loop until condition true (Control Flow)
  • set - Assign variable value (Variables)
  • get - Get variable value (Variables)
  • data - Define inline data (Variables)
  • content - Define reusable content block (Variables)
  • bool - Define boolean variable (Variables)
  • count - Check if array has elements (Counters)
  • increment - Increment variable by 1 (Counters)
  • decrement - Decrement variable by 1 (Counters)
  • echo - Evaluate and output expression (Output)
  • output - Control output buffering (Output)
  • tidy - Clean up whitespace (Output)
  • ignore - Suppress output (Output)
  • open - Output literal { (Output)
  • close - Output literal } (Output)
  • true - Always true condition (Boolean)
  • false - Always false condition (Boolean)
  • null - Null value (Boolean)
  • files - List directory contents (Files)
  • dir - Simple directory listing (Files)
  • file - Write content to file (Files)
  • exists - Check if file exists (Files)
  • page - Include another page (Execution)
  • code - Create execution scope (Execution)
  • sandbox - Isolated execution scope (Execution)
  • pad - Process PAD template (Execution)
  • sequence - Generate numeric sequence (Sequences)
  • pull - Pull value from sequence (Sequences)
  • continue - Skip to next iteration (Sequences)
  • flag - Set iteration flag (Sequences)
  • keep - Keep item in result (Sequences)
  • remove - Remove item from result (Sequences)
  • make - Create new item (Sequences)
  • at - Access item at position (Sequences)
  • curl - HTTP request (Network)
  • action - Form action handler (Network)
  • ajax - AJAX request handler (Network)
  • redirect - Redirect to URL (Network)
  • restart - Restart page processing (Network)
  • dump - Dump debug information (Debug)
  • trace - Output execution trace (Debug)
  • error - Trigger error (Debug)
  • exception - Throw exception (Debug)
  • exit - Stop execution (Debug)
  • field - Get single field value (Database)
  • array - Get multiple rows (Database)
  • record - Get single row (Database)
  • check - Check if record exists (Database)
  • foo - Test/placeholder tag (Misc)
