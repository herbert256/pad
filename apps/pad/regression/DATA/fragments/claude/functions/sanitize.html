The sanitize function provides full special character sanitization.

Basic Usage 

{'<script>alert("test")</script>' | sanitize}

Result: ' ' | sanitize

Compared to HTML 

{'<b>"quoted"</b>' | html}

HTML: '"quoted"' | html 

{'<b>"quoted"</b>' | sanitize}

Sanitize: '"quoted"' | sanitize

Chained Examples 

{'  <dangerous>  ' | trim | sanitize | upper}

Cleaned: ' ' | trim | sanitize | upper

Note

Use sanitize when you need stricter encoding than html. It uses PHP's FILTER_SANITIZE_FULL_SPECIAL_CHARS.
