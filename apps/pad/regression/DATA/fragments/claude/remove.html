The remove tag marks values to exclude during sequence filtering.

Basic Remove 

{sequence '1..10'}
  {if $value eq 5}{remove}{/if}
{/sequence}

This marks value 5 to be removed.

Remove Multiples 

{sequence '1..20', push='nums'}
  {if {echo $value % 3} eq 0}{remove}{/if}
{/sequence}

{pull 'nums', remove}{$value} {/pull}

Remove multiples of 3 from 1-20.

Remove Range 

{sequence '1..10'}
  {if $value >= 4}
    {if $value <= 6}
      {remove}
    {/if}
  {/if}
{/sequence}

Remove values 4, 5, and 6.

Remove vs Keep

Tag Logic Use when
keep Include matching Few to keep
remove Exclude matching Few to remove

Use Case: Filter Out Invalid 

{sequence '1..100', push='data'}
  {if $value < 10}{remove}{/if}
  {if $value > 90}{remove}{/if}
{/sequence}

Remove outliers (less than 10 or greater than 90).

Note

The remove tag is the opposite of keep. Use it to mark values that should be excluded from the final sequence.
