The ajax tag handles AJAX requests.

Syntax

{ajax 'getData'}
  {!-- Returns data for AJAX call --}
  {"status": "ok", "data": [...]}
{/ajax}

JavaScript Integration

<script>
fetch('?ajax=getData')
  .then(r => r.json())
  .then(data => console.log(data));
</script>

{ajax 'getData'}
  {"items": [1, 2, 3]}
{/ajax}

Note

AJAX handlers return only their content, without page wrapper.
