The range function tests if a value is inclusively in a range (min <= value <= max).

Basic Usage 

{$sampleScore | range(0, 100)}

Score 85 in range 0-100: 1

Boundary Test 

{0 | range(0, 100)}

0 in range 0-100: (true - inclusive) 

{100 | range(0, 100)}

100 in range 0-100: (true - inclusive) 

{101 | range(0, 100)}

101 in range 0-100: 101 | range(0, 100) (false)

Grade Ranges 

{$sampleScore | range(90, 100)}

Score 85 is A grade (90-100): 

{$sampleScore | range(80, 89)}

Score 85 is B grade (80-89): 1

Compared to Between

range(0, 100) = inclusive: 0 <= value <= 100

between(0, 100) = exclusive: 0 < value < 100
