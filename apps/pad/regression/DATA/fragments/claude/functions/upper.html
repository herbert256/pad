The upper function converts text to uppercase.

Basic Usage 

{'hello world' | upper}

Result: 'hello world' | upper

Chained Examples 

{$sampleEmail | before('@') | upper}

Username uppercase: JOHN.DOE 

{'  mixed CASE text  ' | trim | upper}

Trim then uppercase: ' mixed CASE text ' | trim | upper

Acronyms 

{'application programming interface' | left(1)}{'application programming interface' | after(' ') | left(1)}{'application programming interface' | afterLast(' ') | left(1)}

Manual acronym: 'application programming interface' | left(1) | upper'application programming interface' | after(' ') | left(1) | upper'application programming interface' | afterLast(' ') | left(1) | upper

With Users 

{users}
  {$status | upper}: {$name}
{/users}

ACTIVE Alice Johnson
PENDING Bob Smith
