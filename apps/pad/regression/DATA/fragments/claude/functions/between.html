The between function tests if a value is exclusively between two values (min < value < max).

Basic Usage 

{$sampleScore | between(0, 100)}

Score 85 between 0 and 100 (exclusive): 1 

{50 | between(0, 100)}

50 between 0 and 100: 50 | between(0, 100)

Boundary Test 

{0 | between(0, 100)}

0 between 0 and 100: (false - not exclusive) 

{100 | between(0, 100)}

100 between 0 and 100: (false - not exclusive)

Age Ranges 

{25 | between(17, 66)}

Age 25 is working age: 25 | between(17, 66) 

{17 | between(17, 66)}

Age 17 (boundary): 17 | between(17, 66) (false - exclusive)

Note

between uses exclusive comparison. For inclusive comparison, use range.
