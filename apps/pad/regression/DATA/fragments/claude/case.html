The case tag provides switch-case style conditional based on value matching.

Basic Case 

{case $color}
  {when 'red'}Color is Red
  {when 'blue'}Color is Blue
  {when 'green'}Color is Green
{/case}

Result ($color = 'blue'): Color is Blue

Case with Default 

{case $day}
  {when 'Saturday'}Weekend!
  {when 'Sunday'}Weekend!
  {when 'default'}Weekday
{/case}

Result ($day = 'Monday'):

Status Example 

{case $status}
  {when 'pending'}<span style="color:orange">Pending</span>
  {when 'approved'}<span style="color:green">Approved</span>
  {when 'rejected'}<span style="color:red">Rejected</span>
{/case}

Result ($status = 'pending'): Pending

Numeric Case 

{case $number}
  {when 1}One
  {when 2}Two
  {when 3}Three
  {when 'default'}Other number
{/case}

Result ($number = 2): Two
