The contains function tests if a string contains a substring.

Basic Usage 

{if {echo $email | contains('@')}}Valid email{/if}

Result: Email contains @

Check for Word 

{if {echo $sampleText | contains('World')}}Has World{/if}

Found: Text contains "World"

Case Sensitive 

{if not {echo 'Hello' | contains('hello')}}Not found{/if}

Note: Case sensitive - 'hello' not in 'Hello'

With Lower 

{if {echo 'Hello' | lower | contains('hello')}}Found{/if}

Case-insensitive: Found after lowercasing

Note

Returns true or false. Use with {if} conditionals.
