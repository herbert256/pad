The dump tag outputs debug information.

Syntax 

{dump}

What It Shows

The dump tag creates a debug dump containing:

  • Current variables and values
  • Request parameters
  • Session data
  • Configuration settings
  • Execution stack trace

Usage 

{if $debug}
  {dump}
{/if}

Note

Dump output goes to DATA/dumps/ directory.

Not demonstrated here to avoid creating dump files.
