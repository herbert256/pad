The output tag sets the output type for PAD processing.

Output Types

PAD supports several output types:

  • web - Default, outputs to browser
  • console - Output for command line
  • file - Write output to file
  • download - Trigger file download

Syntax 

{output 'web'}
{output 'console'}
{output 'file'}
{output 'download'}

Current Output Mode

This page is using the default web output mode.

Example: Download Mode

To create a downloadable file: 

{output 'download'}
{file name='report', ext='txt'}
  Report content here...
{/file}

Example: File Output

To write output to a file instead of displaying: 

{output 'file'}
{file dir='path/to/', name='output', ext='html'}
  <html>...</html>
{/file}

Note

The output tag affects how PAD delivers its final output. Changing it mid-page can have unexpected results. It's typically set at the beginning of processing.
