The curl tag makes HTTP requests.

Syntax

{curl 'https://api.example.com/data'}
  {$response}
{/curl}

Options

OptionDescription
methodHTTP method (GET, POST, etc.)
headersRequest headers
dataPOST data
timeoutRequest timeout

Example POST

{curl 'https://api.example.com/submit', method='POST', data='name=test'}
  Response: {$body}
{/curl}

Note

The curl tag requires network access. Examples here are documentation only.
