The output tag sets the output type.

Output Types

{output 'web'}     - Normal web output (default)
{output 'console'} - Console output
{output 'file'}    - File output
{output 'download'} - Download output

Current Mode

This page uses the default web output mode.

Note

The output tag changes how PAD renders its response. Use 'download' for file downloads, 'console' for CLI output.
