The contains function tests if a string contains a substring (returns boolean).

Basic Usage 

{if {echo $email | contains('@')} eq 1}Valid email format{/if}

Result: Email contains @

Check for Word 

{if {echo $sampleText | contains('World')} eq 1}Contains World{/if}

Found: Text contains "World"

Case Sensitive 

{if {echo 'Hello' | contains('hello')} eq 0}Not found (case matters){/if}

With Lower 

{if {echo 'Hello' | lower | contains('hello')} eq 1}Found{/if}

Case-insensitive match: Found after lowercasing

Note

Returns 1 (true) or 0 (false). Use with {if} conditionals.
