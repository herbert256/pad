The open function outputs a literal opening brace {}.

Basic Usage 

{open}

Result: {

Document PAD Code

Use to show PAD syntax in documentation: 

{open}$variable{close}

Shows: {$variable} 

{open}if $count gt 0{close}

Shows: {if $count gt 0}

Complex Examples 

{open}users{close}
  {open}$name{close}
{open}/users{close}

Shows:

{users}
  {$name}
{/users}

Note

Essential for writing PAD documentation within PAD templates. Without open and close, PAD would try to execute the code instead of displaying it.
