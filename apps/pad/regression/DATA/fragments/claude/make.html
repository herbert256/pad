The make tag transforms values during sequence generation.

Basic Make 

{sequence '1..5'}
  {make}{echo $value * 2}{/make}
{/sequence}

Double each value during iteration.

Transform Values 

{sequence '1..5', push='nums'}
  {make}{echo $value * $value}{/make}
{/sequence}

Squares: {pull 'nums', make}{$value} {/pull}

Create squares of 1-5.

Make with Format 

{sequence '1..5'}
  {make}Item #{$value}{/make}
{/sequence}

Transform numbers into formatted strings.

Mathematical Transformations

Transformation Make Content
Double {echo $value * 2}
Square {echo $value * $value}
Plus 10 {echo $value + 10}
Percentage {echo $value / 100}

Use Case: Data Transformation 

{sequence '0..100', to=10, push='percentages'}
  {make}{$value}%{/make}
{/sequence}

Transform numbers 0, 10, 20... into "0%", "10%", "20%"...

Note

The make tag allows you to transform sequence values during generation. The transformation is applied to each value in the sequence.
