The check tag checks if a record exists.

Syntax 

{check "table where condition"}
  Record exists
{else}
  Record not found
{/check}

Example: User Exists 

{check "users where username = 'admin'"}
  <p>Admin user exists</p>
{else}
  <p>Admin user not found</p>
{/check}

PHP Variable Method 

// In PHP:
$hasAdmin = db("check users where username = 'admin'");

// In template:
{if $hasAdmin}
  Admin exists
{/if}

Use Cases

  • Validate before insert (check duplicates)
  • Authorization checks
  • Feature availability

Note

More efficient than {record} when you only need to know if data exists.
