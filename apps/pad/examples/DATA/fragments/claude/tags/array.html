The array tag retrieves multiple rows from the database.

Syntax

{array "columns from table where condition"}
  {$column1} - {$column2}
{/array}

Example: User List

{array "name, email from users"}
  <tr>
    <td>{$name}</td>
    <td>{$email}</td>
  </tr>
{else}
  <tr><td colspan="2">No users found</td></tr>
{/array}

With Options

{array "* from products", sort='price', rows=10}
  {$name}: ${$price}
{/array}

PHP Variable Method

// In PHP:
$users = db("array * from users");

// In template:
{users}
  {$name}
{/users}

Note

Use &comma; to escape commas in template syntax.

Supports all iteration properties ({first}, {last}, {&count}, etc.)
