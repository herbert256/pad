The right function returns the last N characters.

Basic Usage 

{'Hello World' | right(5)}

Last 5: 'Hello World' | right(5)

File Extension 

{'document.pdf' | right(3)}

Extension: 'document.pdf' | right(3)

Chained Examples 

{'1234567890' | right(4) | + 0}

Last 4 digits as number: '1234567890' | right(4) | + 0 

{$samplePath | right(7) | upper}

Last 7 chars uppercase: ILE.TXT

Complex Chain 

{'Order #ORD-2024-001234' | right(6) | + 1}

Order number + 1: 'Order #ORD-2024-001234' | right(6) | + 1 

{'user@domain.co.uk' | afterLast('.') | right(2) | upper}

Country code: 'user@domain.co.uk' | afterLast('.') | right(2) | upper
