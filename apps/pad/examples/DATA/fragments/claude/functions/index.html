PAD Pipe Functions

Transform values using | function syntax. Chain multiple functions together.

Syntax

{echo $var | function}
{echo $var | function('param')}
{echo $var | func1 | func2 | func3}

Available Functions

  • after - After first delimiter
  • afterLast - After last delimiter
  • before - Before first delimiter
  • beforeLast - Before last delimiter
  • between - Exclusive range test
  • bold - Wrap in bold tags
  • capitalize - Title case
  • close - Literal close brace
  • contains - Contains substring
  • cut - Remove text
  • date - Format date
  • encodeHigh - Encode high ASCII
  • exists - File exists
  • html - HTML escape
  • in - Value in list
  • left - First N characters
  • like - Pattern matching
  • lower - To lowercase
  • max_len - Truncate to length
  • mid - Middle substring
  • nbsp - Non-breaking spaces
  • now - Current timestamp
  • open - Literal open brace
  • optional - Null coalescing
  • range - Inclusive range test
  • replace - Replace text
  • right - Last N characters
  • sanitize - Full sanitization
  • slashes - Add slashes
  • stripLow - Strip control chars
  • stripslashes - Remove slashes
  • substr - Extract by position
  • tag - Create tag syntax
  • time - Format time
  • timestamp - Format timestamp
  • trim - Remove whitespace
  • ucwords - Title case (alias)
  • upper - To uppercase
  • url - URL encode
  • white - Normalize spaces
