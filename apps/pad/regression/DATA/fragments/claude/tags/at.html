The at tag accesses an item at a specific position.

Basic At 

{data 'letters'}["A", "B", "C", "D", "E"]{/data}
{at 'letters', 1}  {!-- First item --}
{at 'letters', 3}  {!-- Third item --}
  • Position 1: A
  • Position 2: A
  • Position 3: A

At with Negative Index 

{at 'letters', -1}  {!-- Last item --}
{at 'letters', -2}  {!-- Second to last --}
  • Last (-1): A
  • Second to last (-2): A

Note

The at tag provides random access to array elements by position.
