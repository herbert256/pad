The action tag handles form actions.

Syntax

{action 'submit'}
  {!-- Executes when action=submit --}
  Form submitted!
{/action}

Form Example

<form method="post">
  <input type="hidden" name="action" value="save">
  <button type="submit">Save</button>
</form>

{action 'save'}
  Data saved!
{/action}

Multiple Actions

{action 'create'}
  Creating...
{/action}

{action 'delete'}
  Deleting...
{/action}

Note

Actions are triggered by POST requests with matching action parameter.
