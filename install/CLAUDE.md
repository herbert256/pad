# install/

Installation scripts for setting up PAD.

## Scripts
- `install.sh` - Main installation script
- `apache.sh` - Apache configuration setup
- `db.sh` - Database setup (creates tables from `database/*.sql`)
- `data.sh` - Data directory setup
- `scripts.sh` - Additional script setup

## Usage
Run these shell scripts to configure a new PAD installation:
```bash
./install/install.sh
```
