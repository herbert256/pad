The trim function removes whitespace from both ends of a string.

Basic Usage 

{$sampleText | trim}

Original: " Hello World "

Trimmed: "Hello World"

Chained Examples 

{'  HELLO WORLD  ' | trim | lower | capitalize}

Clean and format: ' HELLO WORLD ' | trim | lower | capitalize 

{'   john.doe@example.com   ' | trim | before('@') | upper}

Username from padded email: ' john.doe@example.com ' | trim | before('@') | upper

Form Input Cleaning 

{'  user input  ' | trim | lower | replace(' ', '_')}

Sanitized input: ' user input ' | trim | lower | replace(' ', '_')

Complex Chain 

{'   /path/to/file.txt   ' | trim | afterLast('/') | beforeLast('.') | upper}

Filename from padded path: ' /path/to/file.txt ' | trim | afterLast('/') | beforeLast('.') | upper
