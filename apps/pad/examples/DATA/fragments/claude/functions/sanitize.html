The sanitize function performs full sanitization for safe output.

Basic Usage

{echo $htmlText | sanitize}

Result: <p>Hello & "World"</p>

User Input

{echo '' | sanitize}

Safe: <script>bad()</script>

Chained Examples

{echo '  Hello  ' | trim | sanitize}

Clean and safe: <b>Hello</b>

{echo $htmlText | sanitize | left(20)}

Truncated safe: <p>Hello &

Note

Use sanitize for comprehensive protection. It provides more thorough cleaning than html alone.
