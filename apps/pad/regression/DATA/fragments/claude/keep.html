The keep tag marks values to keep during sequence filtering.

Basic Keep 

{sequence '1..10'}
  {if $value > 5}{keep}{/if}
{/sequence}

This marks values greater than 5 to be kept.

Keep Even Numbers 

{sequence '1..10', push='nums'}
  {if {echo $value % 2} eq 0}{keep}{/if}
{/sequence}

{pull 'nums', keep}{$value} {/pull}

Keep only even numbers from 1-10.

Keep with Condition 

{sequence '1..20'}
  {if $value >= 10}
    {if $value <= 15}
      {keep}
    {/if}
  {/if}
{/sequence}

Keep only values between 10 and 15.

Keep vs Remove

Tag Purpose
keep Mark values to include
remove Mark values to exclude
flag Generic marker

Use Case

Use keep when you want to filter a sequence to only include values matching certain criteria. This is part of PAD's sequence filtering system.

Note

The keep tag works during sequence iteration to mark which values should be retained when the sequence is pulled with the keep option.
