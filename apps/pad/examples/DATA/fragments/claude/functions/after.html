The after function returns everything after the first occurrence of a delimiter.

Basic Usage

{echo $email | after('@')}

Result: example.com

Extract Domain

{echo 'user@example.com' | after('@')}

Domain: example.com

Path Extraction

{echo '/home/user/docs/file.txt' | after('/home/')}

Relative path: home/user/docs/file.txt

Chained Examples

{echo $email | after('@') | before('.')}

Domain name only: example

{echo 'Hello: World' | after(': ') | upper}

After colon, uppercase: WORLD

{echo $filePath | after('/') | after('/') | after('/')}

Deep path segment: html/docs/readme.txt

Note

If the delimiter is not found, the original string is returned unchanged.
