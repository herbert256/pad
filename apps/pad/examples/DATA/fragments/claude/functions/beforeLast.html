The beforeLast function returns everything before the last occurrence of a delimiter.

Basic Usage

{echo $filePath | beforeLast('/')}

Directory: /var/www/html/docs

Remove Extension

{echo 'document.backup.txt' | beforeLast('.')}

Without extension: document.backup

Parent Directory

{echo '/var/www/html/index.php' | beforeLast('/')}

Parent: /var/www/html

Chained Examples

{echo $filePath | beforeLast('/') | afterLast('/')}

Parent folder name: /docs

{echo 'a.b.c.d' | beforeLast('.') | beforeLast('.')}

Remove last two: a.b

{echo '/path/to/file.txt' | beforeLast('/') | replace('/path', '') | trim}

Relative dir: /to

Note

Useful for extracting directory paths or removing file extensions.
