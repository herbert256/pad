The url function URL-encodes a string for use in URLs.

Basic Usage

{echo $urlText | url}

Result: hello+world+%26+more

Spaces and Special Chars

{echo 'hello world' | url}

Encoded: hello+world

Query Parameter

{echo 'search term & more' | url}

Safe for URL: search+term+%26+more

Chained Examples

{echo 'My Document.pdf' | url}

Filename: My+Document.pdf

{echo $sampleText | left(10) | url}

Partial encoded: Hello+Worl

Building URLs

?search={echo 'hello world' | url}

Result: ?search=hello+world

Note

Use this when building query strings or encoding values for URLs.
