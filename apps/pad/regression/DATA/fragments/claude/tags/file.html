The file tag writes content to a file.

Syntax 

{file dir='path', name='filename', ext='txt'}
  Content to write
{/file}

Options

Option Description
dir Output directory path
name Base filename
ext File extension
date Append date to filename
stamp Append timestamp
id Append unique ID (length)

Example 

{file dir='output', name='report', ext='txt', date}
  Report content here
  Generated: {date}
{/file}

Creates: output/report_2024-12-14.txt

Note: File writing requires proper directory permissions.
