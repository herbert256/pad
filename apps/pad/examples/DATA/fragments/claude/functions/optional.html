The optional function returns the value or empty string if null (null coalescing).

Basic Usage

{echo $sampleText | optional}

Existing value: Hello World Example Text

{echo $undefinedVar | optional}

Undefined (empty):

Safe Output

Use optional to safely output variables that might not exist:

{echo $maybeNull | optional}

Chained Examples

{echo $sampleText | optional | trim}

Safe trim: Hello World Example Text

{echo $undefinedVar | optional | trim | upper}

Chain on undefined: (empty)

With Default

For default values, use conditional logic instead:

{if $value}{echo $value}{else}default{/if}

Note

The optional function prevents errors when accessing potentially undefined variables. It's similar to PHP's null coalescing operator (??).
