The max_len function truncates text to a maximum length with ellipsis.

Basic Usage 

{echo $sampleText | max_len(15)}

Result: Hello World Exa

Short Text 

{echo 'Hi' | max_len(10)}

Short: Hi

Long Text 

{echo 'This is a very long text that should be truncated' | max_len(20)}

Truncated: This is a very long

Chained Examples 

{echo $sampleText | trim | max_len(12)}

Clean truncate: Hello World 

{echo $email | max_len(15) | html}

Safe display: john.smith@exam

Note

Adds ellipsis (...) when text exceeds the limit. The count includes the ellipsis.
