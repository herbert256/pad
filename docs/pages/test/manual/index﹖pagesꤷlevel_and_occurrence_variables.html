Level and occurrence variables

$xxx variables are set once at the beginning of a level
%xxx variables are set on every occurence inside a level

HTML RESULT 
{set $range = 10}

<table border=1>

  <tr>
    <th>Level</th>
    <th>Occurence</th>
  </tr>

  {sequence range='1..5', $abc=$range, %xyz=$range} 
        <tr>
          <td>{$abc}</td>
          <td>{$xyz}</td>
        </tr>
  {/sequence}

</table>
