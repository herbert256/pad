The pull tag extracts a value from a sequence/iteration.

Basic Pull 

{data 'numbers'}
  [1, 2, 3, 4, 5]
{/data}

{pull 'numbers'} {!-- gets first value --}

First value:

Pull at Position 

{pull 'numbers', 3} {!-- gets 3rd value --}

Third value:

Note

The pull tag is useful for extracting specific values from data without full iteration.
