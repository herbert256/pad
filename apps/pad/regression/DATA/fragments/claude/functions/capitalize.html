The capitalize function converts a string to title case (capitalizes each word).

Basic Usage 

{echo 'hello world' | capitalize}

Result: Hello World

From Lowercase 

{echo 'john smith' | capitalize}

Name: John Smith

From Uppercase 

{echo 'HELLO WORLD' | lower | capitalize}

Fixed: Hello World

Chained Examples 

{echo $email | before('@') | replace('.', ' ') | capitalize}

Name from email: John Smith 

{echo 'the quick brown fox' | capitalize | left(15)}

Truncated title: The Quick Brown 

{echo '  spaced  text  ' | trim | capitalize}

Trimmed and titled: Spaced Text

Note

Alias: ucwords provides the same functionality.
