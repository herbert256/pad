The null tag returns a null/empty value.

Null in Comparison 

{if {null} eq ''}
  Null equals empty string
{/if}

Null equals empty string: true

Null vs Empty String 

{set $empty = ''}
{if $empty eq {null}}
  Empty string equals null
{/if}

Empty string equals null: true

Note

The null tag returns a null value, useful for comparisons and conditionals.
