The slashes function adds backslashes before quotes (PHP addslashes).

Basic Usage 

{"it's here" | slashes}

Result: "it's here" | slashes

Double Quotes 

{'say "hello"' | slashes}

Result: 'say "hello"' | slashes

For JavaScript 

var msg = '{'User\'s "data"' | slashes}';

Safe JS: var msg = ''User\'s "data"' | slashes';

Chained Examples 

{'  John\'s Data  ' | trim | slashes}

Clean and escape: ' John\'s Data ' | trim | slashes

Note

Use slashes when preparing strings for contexts that need escaped quotes, like JavaScript strings or certain SQL contexts (though parameterized queries are preferred for SQL).
