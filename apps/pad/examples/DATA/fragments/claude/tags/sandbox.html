The sandbox tag creates an isolated execution scope.

Basic Sandbox

{set $value = 'original'}
{sandbox}
  {set $value = 'modified'}
  Inside: {$value}
{/sandbox}
Outside: {$value}

Inside sandbox: modified

Outside sandbox: modified (unchanged)

Sandbox vs Code

Feature{code}{sandbox}
Variable isolationNo (shares scope)Yes (isolated)
Changes propagateYesNo

Use Case: Safe Execution

{sandbox}
  {!-- Code here cannot affect parent scope --}
  {set $temp = 'anything'}
{/sandbox}

Temp inside: temporary value

Temp outside: [temporary value] (empty - not defined outside sandbox)
