The false tag returns the boolean FALSE value.

Basic Usage 

{false}

Result:

In Conditions 

{if {false} eq FALSE}
  This is shown because false equals FALSE
{/if}

This is shown because false equals FALSE

Negative Condition 

{if {false} eq TRUE}
  Not shown
{else}
  False is not TRUE
{/if}

False is not TRUE

Set Variable to False 

{set isDisabled={false}}
{if $isDisabled eq FALSE}Enabled{else}Disabled{/if}

Status: Enabled

Toggle Example 

{set flag={true}}
Flag is: {if $flag eq TRUE}ON{else}OFF{/if}

{set flag={false}}
Flag is: {if $flag eq TRUE}ON{else}OFF{/if}

Flag is: OFF

Flag is: OFF

Use Case: Feature Flag 

{set showBetaFeature={false}}
{if $showBetaFeature eq TRUE}
  <div>Beta Feature Here</div>
{/if}

Beta feature is hidden (set to false)
