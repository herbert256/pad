The count tag checks if an array has elements.

Basic Count Check 

{count 'items'}
  Items array has elements
{/count}

Items array has elements

Empty Array Check 

{count 'emptyArray'}
  Has elements
{else}
  Array is empty
{/count}

Array is empty

Count with List 

{count 'users'}
  <p>Users in the system:</p>
  {users}
    - {$name}
  {/users}
{else}
  No users found
{/count}

Users in the system:

  • Alice
  • Bob

Iteration Count Property 

{items}
  Item {&current} of {&count}: {$name}
{/items}
  • Item 1 of 3: Apple
  • Item 2 of 3: Banana
  • Item 3 of 3: Cherry

Note

Use quotes around the array name: {count 'arrayName'}. Without quotes, PAD evaluates the parameter and passes the array itself rather than the name.
