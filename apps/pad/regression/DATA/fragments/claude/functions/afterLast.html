The afterLast function returns everything after the last occurrence of a delimiter.

Basic Usage 

{echo $filePath | afterLast('/')}

Filename: /readme.txt

Get File Extension 

{echo 'document.backup.txt' | afterLast('.')}

Extension: .txt

Last Path Segment 

{echo '/var/www/html/index.php' | afterLast('/')}

File: /index.php

Chained Examples 

{echo $filePath | afterLast('/') | beforeLast('.')}

Filename without extension: /readme 

{echo $filePath | afterLast('/') | upper}

Uppercase filename: /README.TXT 

{echo 'path/to/My File.TXT' | afterLast('/') | lower | replace(' ', '_')}

Normalized: /my_file.txt

Note

Use afterLast to extract filenames from paths or get the last segment of delimited data.
