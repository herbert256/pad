Pipe functions transform values using the | operator. Functions can be chained for complex transformations.

Basic Syntax 

{$variable | functionName}
{$variable | functionName('parameter')}
{$variable | func1 | func2 | func3}

All Functions

  • after - Everything after first delimiter (String Extraction)
  • afterLast - Everything after last delimiter (String Extraction)
  • before - Everything before first delimiter (String Extraction)
  • beforeLast - Everything before last delimiter (String Extraction)
  • substr - Extract substring by position (Substring)
  • left - First N characters (Substring)
  • right - Last N characters (Substring)
  • mid - Substring from position (1-based) (Substring)
  • upper - Convert to uppercase (Case)
  • lower - Convert to lowercase (Case)
  • capitalize - Capitalize first letter of each word (Case)
  • ucwords - Alias for capitalize (Case)
  • trim - Remove whitespace from ends (Manipulation)
  • replace - Replace occurrences of text (Manipulation)
  • cut - Remove all occurrences of text (Manipulation)
  • white - Normalize whitespace (Manipulation)
  • html - Escape HTML special characters (Encoding)
  • sanitize - Full special character sanitization (Encoding)
  • url - URL-encode the value (Encoding)
  • slashes - Add backslashes before quotes (Encoding)
  • stripslashes - Remove backslashes (Encoding)
  • encodeHigh - Encode high ASCII characters (Encoding)
  • stripLow - Strip low ASCII control characters (Encoding)
  • bold - Wrap in <b> tags (HTML)
  • nbsp - Replace spaces with   (HTML)
  • max_len - Truncate to maximum length (Length)
  • contains - Test if value contains substring (Testing)
  • in - Test if value is in list (Testing)
  • like - SQL LIKE pattern matching (Testing)
  • between - Test if exclusively between values (Testing)
  • range - Test if inclusively in range (Testing)
  • exists - Test if file exists (Testing)
  • now - Current Unix timestamp (Date/Time)
  • date - Format timestamp as date (Date/Time)
  • time - Alias for date (Date/Time)
  • timestamp - Alias for date (Date/Time)
  • open - Output literal opening brace (PAD Helpers)
  • close - Output literal closing brace (PAD Helpers)
  • tag - Wrap value as PAD tag (PAD Helpers)
  • optional - Return value or empty if null (PAD Helpers)
