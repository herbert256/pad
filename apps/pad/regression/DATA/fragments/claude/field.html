The field tag retrieves a single field value from the database.

Syntax 

{field 'tablename fieldname'}
{field 'tablename fieldname', where='condition'}

Basic Usage 

{field 'users email'}

Retrieves the email field from the users table.

With Condition 

{field 'users email', where='id=1'}

Retrieves the email for user with id=1.

With Variable 

{field 'users name', where='id=$userId'}

Retrieves the name for the user specified by $userId.

Example Queries

Tag Equivalent SQL
{field 'users name'} SELECT name FROM users LIMIT 1
{field 'users email', where='id=5'} SELECT email FROM users WHERE id=5
{field 'products price', where='sku="ABC"'} SELECT price FROM products WHERE sku='ABC'

Use Cases

  • Retrieve single configuration values
  • Get user information
  • Look up reference data
  • Count or aggregate values

Database Configuration

The field tag requires database configuration in PAD: 

$padSqlHost = 'localhost';
$padSqlDatabase = 'mydb';
$padSqlUser = 'user';
$padSqlPassword = 'pass';

Note

Database queries require a configured database connection. This demo page does not execute live queries.
