The array tag retrieves multiple rows from a database table.

Syntax 

{array 'tablename'}
  {$fieldname}
{/array}

Basic Usage 

{array 'users'}
  <li>{$name} - {$email}</li>
{/array}

Retrieves all rows from users table and iterates over them.

With Options 

{array 'products', sort='price', rows=10}
  {$name}: ${$price}
{/array}

Retrieves products sorted by price, limited to 10 rows.

Common Options

Option Description
sort Sort by field(s)
rows Limit number of results
where Filter condition
order Sort direction (asc/desc)

Example: User List 

<table>
  <tr><th>Name</th><th>Email</th><th>Role</th></tr>
  {array 'users', sort='name'}
    <tr>
      <td>{$name}</td>
      <td>{$email}</td>
      <td>{$role}</td>
    </tr>
  {/array}
</table>

Iteration Properties

Standard iteration properties are available: 

{array 'items'}
  {@first}First item{/first}
  {@current} of {@count}
  {@last}Last item{/last}
{/array}

Comparison with PHP Data

Source Tag
