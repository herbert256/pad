Quick Reference

Tag Syntax 

{tagname}...{/tagname}
{tagname option="value"}
{if $x gt 0}...{/if}

Property Syntax 

{property:first}First item{/property:first}
{property:last}Last item{/property:last}
{property:even}Even row{/property:even}
{property:current} - outputs current index
{property:count} - outputs total count

Pipe Syntax 

{echo $var | function}
{echo $var | function('param')}
{echo $var | trim | upper | left(10)}

Common Patterns 

{echo $email | before('@') | replace('.', ' ') | capitalize}
{echo $path | afterLast('/') | beforeLast('.')}
{echo {now} | date('Y-m-d', '+1 week')}
{items}{property:notFirst}, {/property:notFirst}{$name}{/items}
