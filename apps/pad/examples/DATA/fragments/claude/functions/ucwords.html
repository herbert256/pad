The ucwords function converts a string to title case (alias for capitalize).

Basic Usage 

{echo 'hello world' | ucwords}

Result: Hello World

Name Formatting 

{echo 'jane doe' | ucwords}

Name: Jane Doe

Chained Examples 

{echo 'LOUD TEXT' | lower | ucwords}

Normalized: Loud Text 

{echo $email | before('@') | replace('.', ' ') | ucwords}

Name from email: John Smith

Note

This is an alias for capitalize. Use whichever you prefer.
