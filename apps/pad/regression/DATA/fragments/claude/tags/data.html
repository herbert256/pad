The data tag defines inline data for iteration.

JSON Array 

{data 'colors'}
  [ "red", "green", "blue" ]
{/data}

{colors}
  {$colors}
{/colors}
  • red
  • green
  • blue

JSON Objects 

{data 'products'}
  [
    { "name": "Apple", "price": 1.50 },
    { "name": "Banana", "price": 0.75 }
  ]
{/data}

{products}
  {$name}: ${$price}
{/products}
  • Apple: $1.5
  • Banana: $0.75

Tuple/List Syntax 

{data 'items'}
  ( 'one', 'two', 'three' )
{/data}
  • one
  • two
  • three

PHP Array Iteration

Using $users from PHP: 

{users}
  {$name} ({$role})
{/users}
  • Alice (admin)
  • Bob (user)
  • Carol (user)

Iteration Properties 

{users}
  {first}<ul>{/first}
  <li>#{&current}: {$name}{/li>
  {last}</ul>{/last}
{/users}
  • #1 of 3: Alice
  • #2 of 3: Bob
  • #3 of 3: Carol

Even/Odd Rows 

{users}
  <div style="background:{even}#e0e0e0{/even}{odd}#f0f0f0{/odd}">
    {$name}
  </div>
{/users}
Alice
Bob
Carol
