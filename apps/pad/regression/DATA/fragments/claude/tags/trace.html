The trace tag outputs execution trace information.

Syntax 

{trace}

What It Shows

The trace tag displays:

  • Execution path through templates
  • Tag processing order
  • Variable state changes
  • Timing information

Enable Tracing 

In PHP: $padInfo = 'trace';

Inline Trace 

{trace 'checkpoint 1'}
... some code ...
{trace 'checkpoint 2'}

Note

Useful for debugging complex template execution.
