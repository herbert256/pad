The before function returns everything before the first occurrence of a delimiter.

Basic Usage 

{$sampleEmail | before('@')}

Email: john.doe@example.com

Username: john.doe

First Directory 

{$samplePath | after('/') | before('/')}

Path: /home/user/documents/file.txt

First directory: home

Chained Examples 

{$sampleEmail | before('@') | replace('.', ' ') | capitalize}

Full name from email: John Doe 

{'John Doe ' | before('<') | trim}

Name from email header: 'John Doe ' | before('<') | trim

Complex Chain 

{'2024-12-14T10:30:00' | before('T') | replace('-', '/')}

Date from ISO timestamp: '2024-12-14T10:30:00' | before('T') | replace('-', '/')
