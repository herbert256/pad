The open function outputs a literal opening brace {.

Basic Usage 

{open}

Result: {

Document PAD Syntax 

{open}if $x gt 0{close}

Shows: {if $x gt 0}

Show Variable Syntax 

{open}$username{close}

Shows: {$username}

Show Pipe Syntax 

{open}$name | upper{close}

Shows: {$name | upper}

Complete Example 

{open}users{close}
  {open}$name{close}
{open}/users{close}

Shows:

{users}
  {$name}
{/users}

Note

Use with close to display PAD syntax literally instead of executing it.
