The error tag triggers an error.

Syntax

{error 'Error message'}

Conditional Error

{if $value lt 0}
  {error 'Value cannot be negative'}
{/if}

Validation Example

{if $email eq ''}
  {error 'Email is required'}
{/if}

Note

Error handling depends on $padErrorAction configuration:

  • 'pad' - PAD error page
  • 'php' - PHP error
  • 'stop' - Stop execution
  • 'log' - Log and continue

Not demonstrated to avoid triggering errors.
