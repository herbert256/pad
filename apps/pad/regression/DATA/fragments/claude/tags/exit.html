The exit tag stops page execution.

Syntax 

{exit}

Exit with Message 

{exit 'Goodbye'}

Conditional Exit 

{if $unauthorized}
  <p>Access denied</p>
  {exit}
{/if}

Early Exit Pattern 

{if $done}
  {exit}
{/if}
{!-- This code only runs if not done --}

Note

Exit immediately stops all template processing.

Not demonstrated to avoid stopping page execution.
