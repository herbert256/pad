The slashes function adds backslashes before special characters.

Basic Usage

{echo 'It is a "test"' | slashes}

Result: It is a \"test\"

Escape Quotes

{echo "He said 'hello'" | slashes}

Escaped: He said \'hello\'

Chained Examples

{echo 'Name: "John"' | slashes | left(20)}

Truncated: Name: \"John\"

Note

Use slashes when you need to escape strings for contexts requiring backslash escaping.
