The page tag includes and executes another PAD page.

Basic Page Include 

{page 'fragments/claude/subpage'}

How It Works

The page tag:

  1. Executes the PHP file (pagename.php) to get data
  2. Processes the PAD template (pagename.pad)
  3. Inserts the result into the current page

Variable Scope

Variables from the main page are available in included pages: 

Main page sets: $mainVar = 'Value from main page'
In subpage: {$mainVar}

$mainVar = Value from main page

Include from _include Directory

Files in _include/ can be referenced directly: 

{snippet}
Included Snippet

This content was included from _include/snippet.pad

Include with Variables 

{set userName='Alice'}
{greeting}

Hello, Guest!

Use Cases

  • Modular page components
  • Reusable widgets
  • Include headers/footers
  • Dynamic content sections

Note

The page tag executes both .php and .pad files. For template-only includes, use the _include directory or get tag with stored content.
