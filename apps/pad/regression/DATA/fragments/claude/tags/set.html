The set tag assigns values to variables.

Basic Assignment 

{set $name = 'Alice'}
{set $count = 10}
{set $price = 19.99}
  • Name: Alice
  • Count: 10
  • Price: 19.99

Expression Assignment 

{set $total = $count * $price}
{set $doubled = $count * 2}
  • Total: 199.9
  • Doubled: 20

Increment After Set 

{set $i = 5}
Before: {$i}
{set $i = $i + 1}
After: {$i}

Before: 5

After: 6

Boolean Expression 

{set $isAdmin = (5 gt 3)}

isAdmin: 1
