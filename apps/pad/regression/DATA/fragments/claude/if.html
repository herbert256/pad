The if tag provides conditional execution based on expression evaluation.

Basic If 

{if $count > 0}Has items{/if}

Result: Has items

If with Else 

{if $count eq 0}Empty{else}Has 5 items{/if}

Result: Has 5 items

If-Elseif-Else Chain 

{if $age < 18}Minor{elseif $age < 65}Adult{else}Senior{/if}

Result: Adult

String Comparison 

{if $name eq 'Alice'}Hello Alice!{/if}

Result: Hello Alice!

Not Equal 

{if $name ne 'Bob'}You are not Bob{/if}

Result: You are not Bob

Boolean Check 

{if $isActive eq TRUE}Active{else}Inactive{/if}

Result: Active

Nested If 

{if $count > 0}{if $name eq 'Alice'}Alice has 5 items{/if}{/if}

Result: Alice has 5 items

Comparison Operators

Operator Meaning Example Result
eq / == Equal $count eq 5 true
ne / != Not equal $count ne 0 true
gt / > Greater than $count gt 3 true
lt / < Less than $count lt 10 true
ge / >= Greater or equal $count ge 5 true
le / <= Less or equal $count le 5 true
