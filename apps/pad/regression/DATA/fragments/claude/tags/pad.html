The pad tag processes PAD template content.

Basic PAD Tag 

{pad}
  Template content: {$items} items
{/pad}

Template content processed

PAD with Data 

{data 'myData'}
  [ {"text": "First"}, {"text": "Second"} ]
{/data}

{pad data='myData'}
  @start@
    Item: {$text}
  @end@
{/pad}

Item: First

Item: Second

PAD with Variable Assignment 

{pad $color='blue'}
  Color is: {$color}
{/pad}

Color is: blue

Iteration with @start@ @end@

The @start@ and @end@ markers define the iteration block:

  • First
  • Second
