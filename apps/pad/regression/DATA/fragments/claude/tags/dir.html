The dir tag provides simple directory listing.

Basic Directory 

{dir 'fragments/claude/tags'}
  {$dir}
{/dir}

Directory contents (first 10):

  • .
  • ..
  • _include
  • _lib
  • action.pad
  • ajax.pad
  • array.pad
  • at.pad
  • bool.pad
  • case.pad

Note

The dir tag is simpler than files - it just lists entries without filtering options.

Use files for more control over filtering and file properties.
